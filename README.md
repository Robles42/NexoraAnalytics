# Nexora Analytics Engine

Nexora Analytics es un motor de inteligencia artificial desarrollado para la optimización de inventarios y proyecciones de ventas. Este sistema integra datos de IconicHats y la marca Nexora para predecir quiebres de stock mediante modelos de regresión lineal.

## Tecnologías
* Lenguaje: Python 3.13
* IA/ML: Scikit-Learn (Linear Regression)
* Análisis de Datos: Pandas y Numpy
* Visualización: Matplotlib
* OS: Desarrollado y optimizado para entornos Linux (Fedora/Arch)

## Características Principales
1. Análisis Predictivo: Cálculo de la fecha estimada de agotamiento de stock basada en tendencias de ventas históricas.
2. Visualización Dinámica: Generación de gráficas de tendencia y detección de puntos de quiebre de inventario.
3. Sistema de Alertas: Notificaciones automáticas en terminal para niveles de stock críticos (umbral de 5 días).
4. Persistencia: Gestión de datos de entrada mediante archivos en formato CSV.
